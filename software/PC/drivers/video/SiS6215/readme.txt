// Installation guide for 6215C Rev. 1.04a (5/4/1998)

(1). Windows NT drivers:
     (a). Windows NT 3.5: \VGA\SIS6215\WINNT35\OEMSETUP.INF
     (b). Windows NT 4.0: \VGA\SIS6215\WINNT40\SISV6215.INF

(2). Windows 95 : \VGA\SIS6215\WIN95\SiS215c.inf

(3). Windows 31 drivers and utilities: \VGA\SIS6215\WIN31\SETUP.EXE

(4). OS/2:
     (a). SBCS.21 drivers and utilities: \VGA\SIS6215\OS2\SBCS.21\SISINST.CMD
     (b). SBCS.30 drivers and utilities: \VGA\SIS6215\OS2\SBCS.30\SISINST.CMD
     (c). DBCS.30 drivers and utilities: \VGA\SIS6215\OS2\DBCS.30\SETUP.CMD and
                                         \VGA\SIS6215\OS2\DBCS.30\SISINST.CMD
(5). DOS:
     (a) ADI42 <DIR> : \VGA\SIS6215\DOS\INSTDRV.EXE
          contains AutoCAD, AutoShade, 3D Studio driver files
     (b) ET24  <DIR> : contains ETen Chinese System Display 
          Driver(ETDSPDRV.COM) ver 1.19B01 to support SiS VGA.
          You may update this driver to your ETen Chinese System 
          if you have the older version or you may neglect
          it if you have the newer version.
     (c) UTILITY <DIR> : contains test utilities under DOS.

(6). DirectX5 : Microsoft DirectX5 and Setup programs
