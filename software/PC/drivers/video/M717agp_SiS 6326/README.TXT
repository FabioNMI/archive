
-----------------------------------------------------------
SiS 6326 Driver Installation Guide   Rev. 1.07H (5/27/1998)
-----------------------------------------------------------

SiS 6326 AGP/PCI Graphics & Video Accelerator has five sets of drivers 
for different operation systems as Windows 95, Windows NT, Windows 3.1, 
OS/2 and DOS. The location of these drivers were listed up below.

(1). Windows 95 DirectX , drive and utilities: \VGA\M717AGP\WIN95\SETUP.EXE

(2). Windows NT drivers:
     (a). Windows NT 3.5: \VGA\M717AGP\WINNT\WINNT35\OEMSETUP.INF
     (b). Windows NT 4.0: \VGA\M717AGP\WINNT\WINNT40\SISV6326.INF

(3). OS/2:
     (a). SBCS.21 drivers and utilities: \VGA\M717AGP\OS2\SBCS.21\SISINST.CMD
     (b). SBCS.30 drivers and utilities: \VGA\M717AGP\OS2\SBCS.30\SISINST.CMD
     (c). DBCS.30 drivers and utilities: \VGA\M717AGP\OS2\DBCS.30\SETUP.CMD and
                                         \VGA\M717AGP\OS2\DBCS.30\SISINST.CMD

(4). Windows 31 drivers and utilities: \VGA\M717AGP\WIN31\SETUP.EXE

(5). DOS:
     (a). ADI42 drivers: \VGA\M717AGP\DOS\INSTDRV.EXE
