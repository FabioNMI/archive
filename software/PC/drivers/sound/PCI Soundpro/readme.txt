** PCI Audio Drivers CD Pack Note ** ver 1.99    01/07/98'


\W95-98\drv

WIN95/98 drivers
(supports DirectX 6.0 and Direct sound 3D functions)
(including SoftMIDI, DOS MIDI Emulator & A3D drivers)
If you were installed old drivers,please run UINSTDRV.EXE to uninstall.
Please specify the drivers path when W95/98 system detected sound card
and ask for drivers,W95/98 system will install all of PnP drivers, 
SoftMIDI and A3D drivers at one time. 

\W95-98\app

Audio Rack and C3D HRTF positional audio demo for WIN95/98
Run SETUP.EXE to install applications after Windows drivers installed.

\DOSdrv

DOS drivers
Run INSTALL.EXE to install DOS drivers.

\NT40\drv

Windows NT 4.0 drivers
Please refer to readme file for drivers setup.

User manual
